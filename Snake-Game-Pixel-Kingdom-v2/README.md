# Pixel Kingdom Snake

A modernised Snake game built with Python's standard `turtle` module. The project began as an unfinished learning exercise and was completed with a stable timer-based game loop, scoring, collision detection, progressive difficulty, pause/restart controls, and a colourful retro platformer-inspired interface.

## Features

- Smooth grid-based movement
- Arrow keys and WASD controls
- Food spawning that avoids the snake's body
- Snake growth and score tracking
- Persistent local high score
- Increasing speed as the score rises
- Wall and self-collision detection
- Start, pause, restart, and game-over screens
- Bright red, blue, yellow, green, and brick-inspired visual theme
- No third-party packages or external assets

## Controls

| Action | Key |


| Move | Arrow keys or WASD |

| Start | Space |

| Pause / resume | P |

| Restart | R |

| Exit | Escape |

## Run the game

1. Install Python 3.10 or newer.
2. Open this folder in VS Code.
3. Run `main.py`.

### Windows

```bash
python main.py
```

### macOS or Linux

```bash
python3 main.py
```

`turtle` is included with standard Python installations. On some Linux systems, the separate Tk package may also be required.
